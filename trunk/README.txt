easytc is a simple QT4 based GUI front end for the TrueCrypt application on Linux.

* How to Compile *

You need CMake to compile from the sources.

1. Create a directory named "build" in the top level directory.
2. cd to build directory.
3. Run "cmake .." from there.
4. The "make" to compile. An executable named easytc will be produced
   in the build directory.
